FPC 软板设计

Created2026-03-21|Updated2026-03-21|Learning
|Word Count:1k|Reading Time:3mins|Post Views:

前言

FPC（Flexible Printed Circuit，柔性印刷电路板）因其轻薄、可弯曲的特性，在可穿戴设备、折叠屏手机、无人机等领域得到广泛应用。本文将介绍如何使用嘉立创 EDA 进行 FPC 软板设计。

原理图设计

FPC 的原理图设计与普通 PCB 基本相同，主要区别在于后续的 PCB 布局和工艺要求：

  1. 创建工程：在嘉立创 EDA 中新建工程，选择合适的项目模板
  2. 放置元器件：根据电路需求放置所需元器件
  3. 电气连接：使用导线完成各元器件之间的电气连接
  4. ERC 检查：运行电气规则检查，确保原理图无错误PCB 设计

1. 绘制板框

根据产品结构要求绘制 FPC 板框：

  • 使用 放置 → 板框 工具绘制外形
  • FPC 通常形状不规则，可使用 圆弧、曲线 等工具
  • 注意预留安装孔位和连接器位置

2. 设置 FPC 类型

进入 工具 → 图层管理器，将 PCB 类型改成 FPC 软板：

  • 设置后会多出 顶层补强层 和 底层补强层
  • 补强层用于防止板材在弯折和焊接时损坏

3. 布局与连线

  • 按照信号流向合理摆放元器件
  • 优先布置连接器、按键等关键器件
  • 进行连线，注意走线宽度和间距符合工艺要求
  • 运行 DRC 检查 确保设计规则无误

4. 添加 FPC 补强板

芯片底部最好用钢片，便于焊接

双面胶可以用来粘贴

FPC 补强板是 FPC 设计的关键步骤：

操作方法：

  • 选择 放置 → FPC 补强板
  • 框选需要补强的区域

补强位置：

  • 接口处焊盘：连接器、金手指等插拔区域
  • 焊接区域：需要手工焊接的元器件焊盘
  • 弯折区域：动态弯折部位需要特殊处理

尺寸要求：

  • 补强板应完全包裹住焊盘
  • 边缘超出焊盘 3-5mm 为宜
  • 过大会增加成本，过小保护不足

[!NOTE]

补强层建议放置在焊盘的底层，这样可以在焊接时提供更好的支撑。

5. 铺铜设置

FPC 的铺铜需要特殊设置：

  • 铺铜方式选择 网格铺铜
  • 网格角度设置为 45°
  • 网格线宽和间距根据电流需求调整

45° 网格铺铜的优势：

  • 减少铜箔应力集中
  • 提高 FPC 柔韧性
  • 避免直角走线带来的信号完整性问题

设计检查清单

在下单生产前，请确认以下项目：

  • 板框尺寸符合结构要求
  • 补强层已添加到所有必要区域
  • 走线宽度满足电流承载要求
  • DRC 检查无错误
  • 丝印清晰、方向正确
  • 已添加必要的定位孔

下单生产

1. 生成 Gerber 文件

  • 文件 → 导出 → Gerber
  • 确认所有必要图层已包含
  • 检查 Gerber 预览效果

2. 选择 FPC 工艺

在嘉立创下单时：

  1. 选择 FPC 软板 工艺类型
  2. 确认板材厚度（常用 0.1mm、0.15mm、0.2mm）
  3. 选择表面处理工艺（沉金、沉锡等）
  4. 确认补强片材质和厚度

3. 优惠券

  • 新用户可领取优惠券
  • 关注嘉立创公众号获取活动信息
  • 批量下单可享受更多折扣

常见问题

Q1: FPC 最小弯折半径是多少？

一般建议弯折半径 ≥ 板厚的 6 倍，动态弯折应用需要更大半径。

Q2: 补强层必须添加吗？

接口和焊接区域建议添加，纯走线区域可根据实际情况省略。

Q3: FPC 可以双面贴装吗？

可以，但需要考虑焊接工艺和补强设计，成本会相应增加。

总结

FPC 设计在普通 PCB 设计基础上，需要特别注意：

  1. 板框绘制：适应产品结构
  2. 补强设计：保护关键区域
  3. 铺铜方式：45° 网格铺铜
  4. 工艺选择：下单时选择 FPC 类型
