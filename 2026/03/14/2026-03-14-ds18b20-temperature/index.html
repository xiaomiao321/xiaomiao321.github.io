DS18B20 温度监控模块

Created2026-03-14|Updated2026-03-14|TutorialEmbedded
|Word Count:667|Reading Time:2mins|Post Views:

一、引言

实时监测环境温度是许多智能设备的基础功能之一。本模块使用经典的 DS18B20 单总线数字温度传感器，通过 DallasTemperature 库来获取精确的摄氏度读数，并将历史温度数据绘制成平滑滚动的折线图。

二、实现思路

2.1 双任务协作架构

本模块设计了两个独立的后台任务来协同工作：

  1. updateTempTask：轻量级后台任务，每隔 2 秒钟读取一次温度，更新到全局变量 currentTemperature 中
  2. DS18B20_Task：UI 渲染任务，以更高频率（每 500 毫秒）运行，负责绘制温度数字和折线图

这种”生产者 - 消费者”模式确保了即使传感器响应较慢，UI 刷新也能保持流畅。

2.2 动态图表绘制

TFT_eWidget 库：利用 TFT_eSPI 的扩展库来轻松实现图表绘制。

  • GraphWidget 用于创建图表的背景、坐标轴和网格
  • TraceWidget 用于在图表上绘制折线

滚动图表：程序维护一个 X 轴坐标 gx，每次获取到新的温度点时绘制一个点，然后 gx 自增。当到达右边界时重置为 0 并清空背景，实现无限滚动效果。

2.3 全局温度获取

通过将 updateTempTask 设计为常驻后台任务，并提供全局访问函数 getDS18B20Temp()，项目中任何其他模块都可以随时获取最新的环境温度。

三、核心代码

DallasTemperature 库的使用

#include <OneWire.h>
#include <DallasTemperature.h>

OneWire oneWire(DS18B20_PIN);
DallasTemperature sensors(&oneWire);

void DS18B20_Init() {
  sensors.begin();
}

// 读取温度
void updateTempTask(void *pvParameters) {
    while(1) {
        sensors.requestTemperatures();
        float temp = sensors.getTempCByIndex(0);
        if (temp != DEVICE_DISCONNECTED_C) {
            currentTemperature = temp;
        }
        vTaskDelay(pdMS_TO_TICKS(2000));
    }
}

图表绘制

#include <TFT_eWidget.h>

GraphWidget gr = GraphWidget(&tft);
TraceWidget tr = TraceWidget(&gr);

// 初始化图表
gr.createGraph(WIDTH, HEIGHT, BG_COLOR);
gr.setGraphScale(X_MIN, X_MAX, Y_MIN, Y_MAX);
gr.setGraphGrid(...);
gr.drawGraph(X_POS, Y_POS);

// 开始绘制折线
tr.startTrace(TFT_YELLOW);

// 添加数据点
tr.addPoint(gx, tempC);
gx += 1.0;

// 图表画满后重置
if (gx > 100.0) {
    gx = 0.0;
    gr.drawGraph(TEMP_GRAPH_X, TEMP_GRAPH_Y);
    tr.startTrace(TFT_YELLOW);
}

UI 渲染任务

void DS18B20_Task(void *pvParameters) {
  // 初始化图表和坐标轴
  gr.createGraph(...);
  tr.startTrace(TFT_YELLOW);

  while (1) {
    if (stopDS18B20Task) break;

    float tempC = getDS18B20Temp();

    if (tempC != DEVICE_DISCONNECTED_C) {
      // 绘制数字温度
      // 绘制实时时钟
      
      // 在图表上添加新点
      tr.addPoint(gx, tempC);
      gx += 1.0;

      if (gx > 100.0) {
        gx = 0.0;
        gr.drawGraph(...);
        tr.startTrace(TFT_YELLOW);
      }
    } else {
      // 显示传感器错误
    }
    vTaskDelay(pdMS_TO_TICKS(500));
  }
  vTaskDelete(NULL);
}

四、总结

DS18B20 模块通过双任务架构实现数据采集与 UI 渲染的解耦，使用 TFT_eWidget 库轻松绘制动态温度折线图，全局温度获取函数为其他模块提供便利。
