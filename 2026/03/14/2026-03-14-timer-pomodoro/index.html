一、倒计时模块

1.1 引言

倒计时是工作、学习和生活中非常实用的时间管理工具。本模块实现了一个全功能的倒计时器，用户可以轻松设置分钟和秒数，启动、暂停和继续计时。

1.2 实现思路

状态机驱动的交互逻辑

核心围绕几个关键状态变量构建的状态机：

  • countdown_running: 布尔值，表示计时器是否正在运行
  • countdown_paused: 布尔值，表示计时器是否被暂停
  • countdown_setting_mode: 枚举类型，区分正在设置分钟、秒数或准备就绪

时间管理与计算

  • 时间基准：基于 millis() 函数，记录起始和目标时间
  • 暂停与恢复：通过计算暂停时长并补偿到起始时间，实现精确计时

直观的用户界面

  • 上下文高亮：当前正在被修改的时间单位高亮显示
  • 动态文本：显示”RUNNING”、”PAUSED”等状态
  • 进度条：实时展示时间流逝情况

1.3 核心代码

// 状态变量
static bool countdown_running = false;
static bool countdown_paused = false;
enum CountdownSettingMode { MODE_MINUTES, MODE_SECONDS, MODE_READY_TO_START };

// 暂停处理
if (countdown_running) {
    countdown_pause_time = millis();
    countdown_running = false;
    countdown_paused = true;
}

// 恢复处理
if (countdown_paused) {
    countdown_start_millis += (millis() - countdown_pause_time);
    countdown_running = true;
    countdown_paused = false;
}

// 5 秒预警
if (millis_left > 0 && millis_left <= 5000) {
    tone(BUZZER_PIN, 1000, 100);
}

二、番茄钟模块

2.1 引言

番茄工作法是一种著名的时间管理方法，将工作分解为多个 25 分钟的专注时段，中间穿插短暂的休息。本模块实现了一个完整、自动化的番茄钟。

2.2 实现思路

状态机驱动的循环逻辑

通过 PomodoroState 枚举管理不同状态：

  • STATE_IDLE: 空闲状态
  • STATE_WORK: 25 分钟工作时段
  • STATE_SHORT_BREAK: 5 分钟短休息
  • STATE_LONG_BREAK: 15 分钟长休息
  • STATE_PAUSED: 暂停状态

当一个时段计时结束时，程序会自动切换到下一个状态：

  • 工作结束后，如果已完成 4 次，则进入长休息，否则进入短休息
  • 休息结束后，自动开始下一轮工作

优雅的可视化 UI

  • 圆形进度条：使用 drawSmoothArc 函数绘制
  • 数字时间显示：大号字体清晰显示剩余时间
  • 状态与会话计数：显示当前状态和已完成的专注时段数

2.3 核心代码

enum PomodoroState {
    STATE_IDLE,
    STATE_WORK,
    STATE_SHORT_BREAK,
    STATE_LONG_BREAK,
    STATE_PAUSED
};

// 自动状态流转
if (remaining_ms == 0) {
    tone(BUZZER_PIN, 3000, 3000);
    if (currentState == STATE_WORK) {
        sessions_completed++;
        if (sessions_completed >= SESSIONS_BEFORE_LONG_BREAK) {
            startNewSession(STATE_LONG_BREAK);
        } else {
            startNewSession(STATE_SHORT_BREAK);
        }
    } else {
        startNewSession(STATE_WORK);
    }
}

// 圆形进度条
float progress = (total_secs > 0) ? (float)remaining_secs / total_secs : 0;
int angle = 360 * progress;
menuSprite.drawSmoothArc(120, 120, 100, 80, 0, 360, TFT_DARKGREY, ...);
menuSprite.drawSmoothArc(120, 120, 100-5, 80+5, 0, angle, arc_color, ...);

三、总结

倒计时和番茄钟模块都采用了基于 millis() 的精确计时和状态机驱动的交互逻辑。倒计时模块提供灵活的时间设置和暂停功能，番茄钟模块实现了自动化的工作 - 休息循环，两者都提供了直观的可视化界面。
