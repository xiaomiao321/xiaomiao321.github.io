闹钟模块 - 多组定时与持久化存储

Created2026-03-14|Updated2026-03-14|TutorialEmbedded
|Word Count:822|Reading Time:3mins|Post Views:

一、引言

闹钟是多功能时钟不可或缺的核心功能。本模块实现了一个强大且用户友好的闹钟系统，它允许用户创建、编辑和管理多达 10 组闹钟。所有设置都会被自动保存到 EEPROM 中，确保设备断电后信息不丢失。

二、实现思路

2.1 数据结构与持久化

数据结构：每个闹钟的核心信息被封装在一个 AlarmSetting 结构体中，包含时、分、重复日（使用位掩码表示）、启用状态以及一个用于防止当天重复触发的标志位。

持久化存储：为了实现掉电不丢数据，模块使用了 EEPROM。在初始化时，程序会检查 EEPROM 中是否存有特定的”魔数”。如果存在，则加载已保存的闹钟数据；如果不存在，则初始化一个空的闹钟列表并写入 EEPROM。

2.2 后台检查任务

闹钟的触发逻辑在一个独立的 FreeRTOS 后台任务 Alarm_Check_Task 中运行。该任务每秒执行一次检查函数，负责：

  1. 获取当前时间
  2. 重置每日触发状态（在每天零点过后）
  3. 遍历与匹配所有已启用的闹钟
  4. 如果匹配成功且当天尚未触发过，则触发闹钟

2.3 响铃流程

当闹钟触发时：

  1. 设置全局标志位 g_alarm_is_ringing = true
  2. 创建一个新的后台任务专门用于循环播放提示音乐
  3. 主循环检测到标志后，显示响铃界面
  4. 用户按下按钮即可停止音乐并恢复正常界面

2.4 用户交互界面

闹钟设置 UI 提供了完整的闹钟管理功能：

  • 列表视图：以分页列表形式展示所有闹钟
  • 快速操作：单击可以快速切换闹钟的启用/禁用状态
  • 编辑/添加：双击进入编辑界面，可设置时间、重复日等

EEPROM 读写

static void saveAlarms() {
  EEPROM.write(EEPROM_START_ADDR, EEPROM_MAGIC_KEY);
  EEPROM.put(EEPROM_START_ADDR + 1, alarms);
  EEPROM.commit();
}

static void loadAlarms() {
  if (EEPROM.read(EEPROM_START_ADDR) == EEPROM_MAGIC_KEY) {
    EEPROM.get(EEPROM_START_ADDR + 1, alarms);
  } else {
    // 初始化空数据并保存
  }
}

后台检查任务

void Alarm_Check_Task(void pvParameters) {
    for(;;) {
        Alarm_Loop_Check();
        vTaskDelay(pdMS_TO_TICKS(1000));
    }
}

void Alarm_Loop_Check() {
  // 确保时间已同步
  if (timeinfo.tm_year < 100) return;

  // 每日重置 triggered_today 标志
  static int last_day = -1;
  if (timeinfo.tm_mday != last_day) {
      for (int i = 0; i < alarm_count; ++i) {
          alarms[i].triggered_today = false;
      }
      last_day = timeinfo.tm_mday;
  }

  // 遍历匹配
  for (int i = 0; i < alarm_count; ++i) {
    if (g_alarm_is_ringing) break;
    if (alarms[i].enabled && !alarms[i].triggered_today) {
      bool day_match = alarms[i].days_of_week & (1 << timeinfo.tm_wday);
      if (day_match && alarms[i].hour == timeinfo.tm_hour && 
          alarms[i].minute == timeinfo.tm_min) {
        triggerAlarm(i);
      }
    }
  }
}

响铃界面

void Alarm_ShowRingingScreen() {
    // 绘制 "Time's Up!" 等 UI 元素
    while (g_alarm_is_ringing) {
        if (readButton()) {
            Alarm_StopMusic();
        }
        vTaskDelay(pdMS_TO_TICKS(50));
    }
}

四、总结

闹钟模块通过位掩码高效存储重复日设置，使用 EEPROM 实现持久化存储，后台检查任务确保准时触发，响铃流程提供完整的声光提醒。
