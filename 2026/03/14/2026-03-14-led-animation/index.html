LED 与动画模块 - NeoPixel 灯带控制

一、LED 控制模块

1.1 引言

本模块为用户提供了一个直接、手动的界面来控制 NeoPixel（WS2812B）可寻址 LED 灯带。用户可以通过旋转编码器精确地调节灯带的整体亮度和颜色。

1.2 实现思路

双模式控制：通过 ControlMode 枚举决定当前旋转编码器的输入应该用于调整亮度还是颜色。

HSV 颜色空间：通过控制 HSV 颜色空间中的 H（色相）值来改变颜色，strip.ColorHSV(hue) 函数将 0-65535 范围的色相值转换为 RGB 颜色。

可视化反馈：使用两个水平进度条分别展示亮度和颜色，颜色条的填充色实时根据当前色相变化。

1.3 核心代码

#include <Adafruit_NeoPixel.h>

Adafruit_NeoPixel strip = Adafruit_NeoPixel(NUM_LEDS, LED_PIN, NEO_GRB + NEO_KHZ800);

enum ControlMode { BRIGHTNESS_MODE, COLOR_MODE };

void LEDMenu() {
    ControlMode currentMode = BRIGHTNESS_MODE;
    uint8_t brightness = strip.getBrightness();
    uint16_t hue = 0;

    while (true) {
        int encoderChange = readEncoder();
        
        if (currentMode == BRIGHTNESS_MODE) {
            brightness += (encoderChange * 10);
            strip.setBrightness(brightness);
        } else {
            hue += (encoderChange * 2048);
        }

        if (readButton()) {
            if (led_detectDoubleClick()) {
                break; // 双击退出
            } else {
                currentMode = (currentMode == BRIGHTNESS_MODE) ? COLOR_MODE : BRIGHTNESS_MODE;
            }
        }

        // 更新灯带
        strip.fill(strip.ColorHSV(hue));
        strip.show();
        
        vTaskDelay(pdMS_TO_TICKS(20));
    }
}

二、动画模块

2.1 引言

动画模块创建了一个独立的后台任务，用于生成不断变化的随机动画，同步控制 NeoPixel 灯带颜色，并配以动态蜂鸣器音效。

2.2 实现思路

后台任务驱动：核心逻辑封装在独立的 FreeRTOS 任务 Animation_task 中，不阻塞主 UI 线程。

随机动画生成：每一帧由随机参数生成：

  • 使用 drawSmoothArc 绘制抗锯齿平滑圆弧
  • 从 TFT 的 16 位色提取 RGB 分量转换为 NeoPixel 的 24 位色
  • 使用 tone() 播放随机频率声音

动画节奏控制：动画速度从 500ms 逐渐减小到 50ms，创造动态节奏感。

2.3 核心代码

void Animation_task(void *pvParameters) {
  int delay_ms = 500;

  while(!stopAnimationTask) {
    // 1. 绘制动画帧
    uint16_t fg_color = random(0x10000);
    int x = random(50, 190);
    int y = random(50, 190);
    int radius = random(30, 80);
    
    tft.drawSmoothArc(x, y, radius, radius-10, 
                      random(0, 360), random(0, 360), 
                      fg_color, bg_color, arc_end);

    // 2. 更新 NeoPixel（16 位转 24 位）
    uint8_t r = (fg_color & 0xF800) >> 8;
    uint8_t g = (fg_color & 0x07E0) >> 3;
    uint8_t b = (fg_color & 0x001F) << 3;
    strip.fill(strip.Color(r, g, b));
    strip.show();

    // 3. 播放声音
    tone(BUZZER_PIN, random(800, 1500), delay_ms);

    // 4. 控制速度
    vTaskDelay(pdMS_TO_TICKS(delay_ms));
    if (delay_ms > 50) delay_ms -= 10;
  }

  // 清理
  noTone(BUZZER_PIN);
  strip.clear();
  strip.show();
  vTaskDelete(NULL);
}

2.4 颜色格式转换

// TFT 屏幕：16 位 RGB565
uint16_t fg_color = random(0x10000);

// NeoPixel: 24 位 RGB888
uint8_t r = (fg_color & 0xF800) >> 8;
uint8_t g = (fg_color & 0x07E0) >> 3;
uint8_t b = (fg_color & 0x001F) << 3;

strip.fill(strip.Color(r, g, b));

三、总结

LED 控制模块通过 HSV 颜色空间实现直观的颜色调节，动画模块通过后台任务实现声光画同步的随机艺术效果，颜色格式转换实现 TFT 与 NeoPixel 的颜色同步。
