Created2026-03-14|Updated2026-03-14|TutorialEmbeddedUI

一、引言

表盘是时钟模块的核心视觉展示部分。本模块实现了 17 款风格各异的时钟表盘，每款都有独特的视觉效果和动画。用户可以在表盘菜单中选择自己喜欢的样式。

二、表盘列表

矢量扫描系列

  1. VectorScan：矢量扫描风格，简洁线条显示时间
  2. VectorScan_SEG：矢量扫描 + 数码管风格
  3. VectorScroll：矢量滚动效果
  4. VectorScroll_SEG：矢量滚动 + 数码管风格

进度条系列

  1. ProgressBar：使用进度条展示时间流逝

模拟时钟系列

  1. SimClock：经典模拟时钟，带时分秒指针
  2. SimpleClock：极简风格时钟

特效系列

  1. Galaxy：银河旋转效果
  2. TerminalSim：终端模拟器风格
  3. CodeRain：黑客帝国代码雨效果
  4. Snow：雪花飘落效果
  5. Waves：波浪动画效果
  6. Neno：霓虹灯效果
  7. Balls：弹跳小球效果
  8. SandBox：沙盒粒子效果
  9. Cube3D：3D 立方体旋转
  10. Charge：充电风格显示

3.3 代码雨效果示例

void Watchface_CodeRain() {
    // 初始化列位置和字符
    int columns = SCREEN_WIDTH / FONT_WIDTH;
    int drops[columns];
    
    while(true) {
        if (exitSubMenu) return;
        
        // 半透明黑色覆盖，产生拖尾效果
        menuSprite.fillRect(0, 0, SCREEN_WIDTH, SCREEN_HEIGHT, 
                           color565(0, 0, 0, 20));
        
        // 绘制字符
        for (int i = 0; i < columns; i++) {
            char c = random(0x30, 0x7A); // 随机字符
            menuSprite.drawString(String(c), i * FONT_WIDTH, 
                                 drops[i] * FONT_HEIGHT, TFT_GREEN);
            drops[i]++;
            
            // 随机重置
            if (drops[i] * FONT_HEIGHT > SCREEN_HEIGHT && 
                random(100) > 95) {
                drops[i] = 0;
            }
        }
        
        menuSprite.pushSprite(0, 0);
        vTaskDelay(pdMS_TO_TICKS(50));
    }
}

3.4 银河旋转效果

void Watchface_Galaxy() {
    float angle = 0;
    
    while(true) {
        if (exitSubMenu) return;
        
        menuSprite.fillSprite(TFT_BLACK);
        
        // 绘制中心时间
        strftime(timeStr, sizeof(timeStr), "%H:%M:%S", &timeinfo);
        menuSprite.drawString(timeStr, 120, 120, TFT_WHITE);
        
        // 绘制旋转的银河臂
        for (int i = 0; i < 100; i++) {
            float starAngle = angle + (i * 0.1);
            float radius = i * 0.5;
            int x = 120 + cos(starAngle) * radius;
            int y = 120 + sin(starAngle) * radius;
            menuSprite.drawPixel(x, y, TFT_BLUE);
        }
        
        angle += 0.02;
        menuSprite.pushSprite(0, 0);
        vTaskDelay(pdMS_TO_TICKS(30));
    }
}

四、总结

表盘系统提供了 17 款风格各异的时钟显示样式，从简洁的矢量扫描到炫酷的代码雨、3D 立方体，每款表盘都是独立的函数模块，易于扩展和维护。用户可以根据个人喜好和使用场景选择合适的表盘。
