音乐播放器模块 - 蜂鸣器与 LED 灯效

Created2026-03-14|Updated2026-03-14|TutorialEmbedded
|Word Count:711|Reading Time:2mins|Post Views:

一、引言

本模块将简单的无源蜂鸣器变成一个功能丰富的音乐播放器。它能够播放多首预先定义好的单音旋律，并配有同步的 LED 彩虹灯效，创造出动感的视听体验。

二、实现思路

2.1 歌曲数据的存储

PROGMEM：单片机的 RAM 资源非常有限，将大量的歌曲数据直接定义为普通数组会迅速耗尽 RAM。因此，本模块使用 PROGMEM 关键字将这些静态、只读的数据存放在程序闪存中。

数据结构：每首歌曲被定义为一个 Song 结构体，包含歌曲名、指向音高数组的指针、指向时长数组的指针和歌曲长度。

typedef struct {
  const char* name;
  const int* melody;
  const int* durations;
  int length;
} Song;

const Song songs[] PROGMEM = {
  { "Cai Bu Tou", melody_cai_bu_tou, durations_cai_bu_tou, length },
  // ... 更多歌曲
};

2.2 多任务播放架构

音乐播放和灯光效果都在独立的后台任务中运行：

  • Buzzer_Task：核心音乐播放任务，根据播放模式循环播放旋律
  • Led_Rainbow_Task：独立生成平滑过渡的彩虹灯效

2.3 播放界面与可视化

  • UI 状态机：歌曲选择列表和播放界面两个阶段
  • 实时进度显示：计算并显示已播放时长和总时长
  • 模拟频谱：根据当前音符频率生成模拟频谱数据

三、核心代码

PROGMEM 的使用

Song song;

// 从 PROGMEM 中读取音高和时长
int note = pgm_read_word(song.melody + i);

// 播放音符
tone(BUZZER_PIN, note, duration * 0.9);

音乐播放任务

void Buzzer_Task(void *pvParameters) {
  int songIdx = *(int*)pvParameters;
  
  for(;;) {
    Song song;
    memcpy_P(&song, &songs[songIdx], sizeof(Song));
    
    for (int i = 0; i < song.length; i++) {
      while (isPaused) { vTaskDelay(pdMS_TO_TICKS(50)); }
      
      int note = pgm_read_word(song.melody + i);
      int duration = pgm_read_word(song.durations + i);
      
      generate_fake_spectrum(note);
      if (note > 0) {
        tone(BUZZER_PIN, note, duration * 0.9);
      }
      vTaskDelay(pdMS_TO_TICKS(duration));
    }
    
    // 根据播放模式选择下一首
    if (currentPlayMode == LIST_LOOP) {
      songIdx = (songIdx + 1) % numSongs;
    }
  }
}

LED 彩虹灯效

void Led_Rainbow_Task(void *pvParameters) {
    for (uint16_t j = 0; ; j++) {
        if (stopLedTask) {
            strip.clear();
            strip.show();
            vTaskDelete(NULL);
        }
        for (uint16_t i = 0; i < strip.numPixels(); i++) {
            strip.setPixelColor(i, Wheel(((i * 256 / strip.numPixels()) + j) & 255));
        }
        strip.show();
        vTaskDelay(pdMS_TO_TICKS(20));
    }
}

模拟频谱生成

void generate_fake_spectrum(int frequency) {
    for (int i = 0; i < NUM_BANDS; i++) { spectrum[i] = 0.0f; }
    if (frequency <= 0) return;
    
    // 对数频率映射
    float log_freq = log10(frequency);
    int band = NUM_BANDS * (log_freq - min_log_freq) / (max_log_freq - min_log_freq);
    
    // 在目标频段设置幅度
    spectrum[band] = 70.0f;
    spectrum[band - 1] = 40.0f;
    spectrum[band + 1] = 40.0f;
}

四、总结

本模块通过 PROGMEM 高效存储歌曲数据，多任务架构实现音乐播放与灯效同步，模拟频谱提供视觉反馈。内置 88 首歌曲，支持单曲循环、列表循环和随机播放三种模式。
