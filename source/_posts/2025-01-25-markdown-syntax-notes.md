## 一、基本语法

### 1. 标题

````markdown
# 一级标题
## 二级标题
### 三级标题
````

### 2. 加粗

```markdown
**要加粗的**
```

**要加粗的**

没加粗的

或者按下 `control` + `B`

### 3. 斜体

```markdown
*要斜体的*
```

*要斜体的*

没斜体的

或者按下 `control` + `I`

### 4. 删除线

```markdown
~~要删除的~~
```

~~要删除的~~

### 5. 高亮

```markdown
==要高亮的==
```

==要高亮的==

### 6. 代码

```markdown
`一个代码`
```

`一个代码`

### 7. 代码框

````markdown
```c
int main()
{
    for(int i = 0;i<n;i++)
    return 1;
}
```
````

`esc` 下面那个按键

### 8. 引用

```markdown
> 引用文本 (可以嵌套)
```

> 引用文本

>> 111

### 9. 无序列表

```markdown
- 第一行
```

- 第一行
- 按下 `enter` 后自动出列表黑点
  - 按下 `tab` 后变成空心的
- 按下 `shift`+`tab` 又可以变回来

### 10. 有序列表（数字+"."+空格）

```markdown
1. 111
2. 222
3. 333
```

1. 111
2. 222
3. 333

### 11. 上标

```markdown
x^上标^
```

x^上标^

### 12. 下标

```markdown
x~下标~
```

x~下标~

### 13. 注释

```markdown
正文 [^1]
[^1]:注释
```

正文 [^1]
[^1]:注释（应该写在文档最后）

### 14. 链接（常用"[]"+"()" 分别包围文本和链接）

```markdown
[百度](www.baidu.com)
[1. 标题](#1.标题)
```

[百度](www.baidu.com)

[1. 标题](#1.标题)

支持网页链接和文内跳转，按住 `control` + `单击鼠标左键` 即可跳转

### 15. 任务列表 ("- [ ]" + 空格)

```markdown
- [ ] 111
```

- [ ] 111

### 16. 表格（用"\|" 绘制表格边框）

```markdown
| 学号 | 姓名 | 年龄 |
| :--- | :---: | ---: |(引号的位置代表左对齐、居中、右对齐)
第一行为表头，并由第二行分割线决定对齐方式与长度，第三行之后即表格数据
```

| 学号 | 姓名 | 年龄 |
| :--: | :---- | :--: |
| 111 | 明太鱼 |111 |
| 222|鲈鱼|222|

分割线:`---`
