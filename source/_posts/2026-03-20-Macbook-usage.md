## 使用差异

### 界面操作逻辑

| Windows | MacOS |
|---------|-------|
| 此电脑 | 访达 |
| 回收站 | 废纸篓 |

关闭按钮在左上角

> [!NOTE]
> 关闭按钮只是关闭了当前窗口，但是软件仍在后台运行

想要退出软件，可以：

1. `Command` + `Q`
2. 菜单栏退出
3. 右键程序坞对应软件图标退出

**全屏**：不同于 Windows 的最大化，而是会隐藏菜单栏和程序坞，相当于显示在一个新的桌面，此时可以通过三指滑动来切换桌面

**分屏**：

1. 两个程序都已经全屏：四指上推，然后三指将需要用到的两个桌面拖动到一起，两个桌面会自动结合
2. 两个程序未全屏：点击绿色全屏按钮 → 将窗口拼贴到屏幕左侧

**最大化窗口**：

1. 按住键盘 `Option`，再点全屏按钮
2. 双击软件标题栏
3. 双击软件窗口边缘

### 输入设备

#### 鼠标与触控板

- **官方鼠标**：Magic Mouse（无物理滚轮）
- **普通鼠标**：滚动方向是反的
  - 解决：系统设置 → 触控板 → 滚动缩放 → 关闭「自然滚动」
  - 建议下载 [Mos](https://mos.caldis.me/) 优化滚动体验

**触控板手势**：

| 手势 | 功能 |
|------|------|
| 轻点 | 点按（需在设置中开启） |
| 三指向上轻扫 | 打开调度中心 |
| 拇指与其余三指合拢 | 打开启动台 |
| 拇指与其余三指张开 | 显示桌面 |

#### 键盘

左下角按键不同于 Windows，依次是：`fn`、`control`、`option`、`command`，大部分快捷键与 `Command` 有关。

- **Windows 键盘**：接入后可通过 `Fn` + `M` 打开 Mac 模式
- **自定义修饰键**：设置 → 键盘 → 修饰键

### 常用快捷键

#### 基础操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Command` + `A` | 全选 | |
| `Command` + `C` | 拷贝 | 和 Windows 的复制一样 |
| `Command` + `V` | 粘贴 | |
| `Command` + `X` | 剪切 | 仅支持文本，不支持文件 |
| `Command` + `D` | 复制 | 生成一个相同的副本在旁边 |

#### 窗口管理

| 快捷键 | 功能 |
|--------|------|
| `Command` + `W` | 关闭当前窗口 |
| `Command` + `Q` | 关闭当前程序 |
| `Command` + `M` | 最小化当前窗口 |
| `Command` + `H` | 隐藏当前程序 |
| `Command` + `Option` + `H` | 隐藏所有后台程序 |
| `Command` + `N` | 新建当前程序的文件 |
| `Command` + `,` | 打开偏好设置 |

#### 应用切换

| 快捷键 | 功能 |
|--------|------|
| `Command` + `Tab` | 切换到下一个应用程序 |
| `Command` + `Shift` + `Tab` | 切换到上一个应用程序 |
| `Command` + `~` | 在同一程序的不同窗口间切换（Safari 使用频率高） |

#### 文件操作

| 快捷键 | 功能 |
|--------|------|
| `Command` + `C` → `Command` + `Option` + `V` | 文件剪切 | 先复制，再「移动」粘贴 |
| `空格` | 快速预览文件 | |

#### 截图

| 快捷键 | 功能 |
|--------|------|
| `Shift` + `Command` + `3` | 全屏截图 |
| `Shift` + `Command` + `4` | 框选截图 |
| `Shift` + `Command` + `4` + `空格` | 窗口截图 |
| `Shift` + `Command` + `5` | 打开截图工具，可选择截图方式 |

#### 其他

| 快捷键 | 功能 |
|--------|------|
| `Command` + `Space` | 聚焦搜索 |
| `Control` + `空格` | 切换输入法 |
| `Command` + `L` | Safari 浏览器网址栏聚焦 |
| `Command` + `P` | 打印 |
| `Command` + `Shift` + `P` | 页面设置 |

## 软件安装

### 安装方式

1. **官网下载**：下载 `.dmg`（Disk Image）文件
2. **App Store**：直接在应用商店下载

### 安装步骤

1. 打开 `.dmg` 文件
2. 将应用拖到 `Applications` 文件夹
3. 退出临时挂载的磁盘

### 允许安装第三方应用

有时系统会限制安装非 App Store 应用：

1. 打开 系统偏好设置 → 安全与隐私 → 允许任何来源
2. 若没有该选项，在终端输入：

```bash
sudo spctl --master-disable
```

> [!TIP]
> 推荐工具：DaisyDisk（磁盘清理）、柠檬清理

## 自定义菜单栏

- **调整顺序**：按住 `Command` 拖动菜单栏图标可改变顺序
- **删除图标**：按住 `Command` 拖动图标到菜单栏外即可删除
